## 변경 사항 상세

### 1) 카테고리 차트 라벨 — MLB 표준 명시

**Before**:
```
[고교 또래 대비 내 수준]    체력           [85점]
                                          /100 percentile
※ 같은 고교 코호트 중앙값 = 50점, 상위권 = 70점+
```

**After**:
```
[MLB·문헌 표준 대비 발달도]    체력           [85점]
                                              /100
※ 100점 = MLB elite 표준값. 80점+ = 한국 고1 elite 적정. 50점 = 발달 평균.
```

### 2) 종합 평가 또래 percentile 행 — 단기 향상 가이드 강화

```
📍 한국 고교 코호트 percentile — 단기 향상 가이드
※ 위 점수(MLB·문헌 표준)는 장기 목표 지표,
  아래 percentile은 한국 고교 또래 대비 현재 위치 — 단기 향상의 현실적 출발점.

체력      41.5점   13/13 변수 측정 (100%)   신뢰도 높음
메카닉    59점     12/27 변수 측정 (44%)    신뢰도 낮음 ⚠
제구      —점      8/11 변수 측정 (73%)    신뢰도 중간
```

### 3) 5단계 트레이닝 우선순위 — 결함별 코호트 비교 추가 (NEW)

각 결함 카드에 **한국 고교 코호트 비교** 자동 표시:
```
우선순위 1 · 단계 2 앞다리 블록 · 앞무릎 무너짐 ⚠
코칭 방향: block 다리 stiff block 만들기, 무릎 펴진 자세 유지
📍 한국 고교 코호트 비교 (단기 향상 가이드)
   본인 -15° · 또래 평균 -8° · 코호트 상위 25% 0°
   → 단기 목표: 또래 상위 25% 진입 (0°)
💪 추천 드릴:
   1) Eccentric step-down 5초 3×6
   2) Single-leg RDL
   3) Glute bridge ISO 30s
```

**FAULT_COHORT_VAR 매핑**:
- WeakDrive → max_cog_velo
- LeadKneeCollapse → lead_knee_ext_change_fc_to_br
- SoftBlock → com_decel_pct
- TrunkArmDisconnect → arm_trunk_speedup
- ElbowOverload → elbow_ext_vel_max
- ... (16개 결함 ↔ 코호트 변수)

### 4) 5단계 헤더 메시지

```
💪 5단계 · 어떻게 개선할까
발견된 N개 결함 중 우선순위 X개를 집중 트레이닝 권장합니다.
단기 향상 목표는 한국 고교 코호트 기반이며, 장기 목표는 카테고리 차트의 MLB·문헌 표준 점수입니다.
```

## 의도된 코칭 흐름

1. **헤더 카드**: 측정 구속 + 잠재 구속 (즉각 동기부여)
2. **카테고리 차트** (MLB·문헌 표준): 장기 목표 — "이게 도달할 수준"
3. **종합 평가 → 또래 percentile**: 단기 위치 — "현재 또래 어디인가"
4. **5단계 트레이닝 우선순위 (코호트 비교)**: 즉각 행동 — "다음 단계로 가려면 이 변수를 X에서 Y로"
5. **결함 진단 + 드릴**: 구체 실천
