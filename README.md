# BBL Pitcher Integrated Report — 샘플 분석 보고서

국민대학교 **BioMotion Baseball Lab (BBL)** 투수 통합 분석 리포트 공개 버전입니다.

- **측정** · Uplift Labs 마커리스 모션캡처 · VALD ForceDecks · 랩소도
- **분석 대상** · 김주원 · 홍주환 · 정원진 · 권준서 (샘플 4명)
- **홈** · https://biomotion.kr

## 배포 방법 (GitHub Pages)

1. 이 폴더 전체를 GitHub 저장소에 **Public**으로 업로드합니다.
2. 저장소 → **Settings → Pages → Source: `main` branch / root** → Save.
3. 약 1~2분 후 `https://<사용자명>.github.io/<저장소명>/` 에서 접속 가능합니다.
